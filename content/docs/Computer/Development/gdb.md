linkTitle: 调试
title: 程序调试
---

1. 编译时加入 -g 调试选项
2. 运行 gdb [程序名称]
3. 在 gdb 中运行 run [程序参数]

- 程序崩溃
  - 崩溃后运行 bt
- 添加断点
  - 运行 b [源文件]:[行号]
- 删除所有断点
  - 运行 d
- 查看变量
  - p [变量名]
  - p [数组地址]@[打印元素数目]
- 切换栈帧
  - 运行 up 切换到上一级栈帧
  - 运行 down 切换到下一级栈帧
- 查看源码
  - 运行 list
  - 分屏查看
    - 运行 layout src
    - 通过 Ctrl+x o 在源码和命令行窗口之间切换
    - 通过 Ctrl+x a 退出源码界面
