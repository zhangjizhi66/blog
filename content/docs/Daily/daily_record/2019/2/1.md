title: 2019年2月1日
---

上午的行程是去机场附近的红树林。工作人员给我们讲了很多红树的知识，我们亲手种下了小树苗，还在滩涂上看到了螃蟹交配。中途突然下起大雨。

中午吃了简单的炒饭，z 和 c 退了房，来我房间午休、放行李。原本预约的 spa 都满了，我们临时决定改去玩皮划艇。

下午天放晴了，我们换上泳衣上了船。我和 c、s和z 分在一组，水很浅，底下的石头硌得脚生疼。我带着手机密封套下水拍照，上船时费了好大力气。我们故意 “把船打横” “就是要体会翻船的感觉” “翻了就翻了吧”。结果浪真的太大了，c 第一个落水，吓得紧紧抓着船边，紧接着又一个浪打过来，船彻底翻了，我也掉进了水里。我的泳镜和盒子沉了下去，我拼命去捞，结果摸上来的是挂在脖子上的手机，桨也丢了一只。等了好几波浪过去，c 先爬上船，我刚上去，船又翻了，膝盖磕在海底的石头上，破了皮。好不容易爬回船上往回划，太阳出来了，皮肤被晒得发红，还有骑摩托艇的老外围着我们转圈，故意溅起水花。z 和 s 在我们翻船前就划远了，后来也翻了船。c 说这是她四年级之后第一次下海，吓得不轻。

丢桨的事没被发现，z 和 s 后来租了摩托艇玩，我陪 c 在沙滩上坐着聊天。六点回到宾馆，z 和 c 去机场，我和 s 在酒店吃了最后一顿晚餐，九点，我们也分别了。

热闹了几天，突然只剩我一个人了。我开始安排明天去佩尼达岛的行程，和张佳燕、许佳悠一起，费用 328。
